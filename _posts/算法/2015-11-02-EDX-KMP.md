1.字符串
定义:由来自字母表的字符所组成的有限序列
特性字符的种类不多，而串的长度远大于字母表包含的种类数量(比如英文文章)

2.术语
字符串相等:长度相等,且对应的字符均相同
子串(sub string):连续k个字符
前缀(prefix):前k个字符
后缀(suffix):后k个字符
空串:任何串的子串,前缀,后缀
约定:任何串都是自己的子串,前缀,后缀

3.抽象数据类型(ADT,abstract data type)
length()
charAt()
substr(i,k)
prefix(k)
suffix(k)
concat(T) (串接)
equal(T)
indexOf(P) (string S是否有子串与P完全相等)

4.串匹配问题
(1)grep命令 <pattern> <text>
(2)text >> pattern (text远大于pattern)
(3)匹配功能(pattern matching):
-detection:   P是否出现?      (只关注是否出现, e.g.搜索病毒关键码)
-location:    首次在哪里出现? (主要问题)
-counting:    共有几次出现?
-enumeration(枚举): 各出现在哪里?

6.蛮力匹配(brutal force)
(1)逐一尝试匹配
(2)自左向右,以字符为单位移动模式串,直到发现匹配.

7.蛮力匹配(brutal force)代码 (版本1)

```
int match(char * P, char * T){  //尾部有'\0'
	size_t n = strlen(T), i = 0;
	size_t m = strlen(P), j = 0;
	while( j < m && i < n){  //自左向右逐个比对字符, j == m的时候,匹配成功; i == n的时候,匹配完毕
		if( T[i] == P[j] ) {
			i++; j++;        //若匹配,则转到下一对字符
		} else{
			//否则,T回退,P复位	  
			i -= j;		 
			j =  0;
			//再更新
			i++;
		}
	}
	
	return i - j; //在text中匹配到的位置, 假如i-j > n-m, 失败
}
```
